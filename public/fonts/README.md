# Font Setup — Frama (Pangram Pangram)

1. Purchase/download Frama from https://pangrampangram.com/products/frama
2. Place the following .woff2 files here:
   - Frama-Light.woff2
   - Frama-Regular.woff2
   - Frama-Medium.woff2
   - Frama-SemiBold.woff2
   - Frama-Bold.woff2
3. The font is already configured in `app/layout.tsx` via `next/font/local`
